#  -----------------------------------------------
#  1. Open Command Prompt (Win+R → type "cmd" → Enter)
#
#  2. Navigate to your project folder:
#        cd D:\Jefrin\BTr\batch_tracker_v001_starter
#
#  3. Activate your virtual environment:
#        .venv\Scripts\activate
#     You should see (.venv) appear at the start of the line.
#
#  4. Install PyTorch FIRST (needs special command):
#        pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
#     NOTE: cu121 = CUDA 12.1. If your GPU driver supports a different
#     CUDA version, check https://pytorch.org/get-started/locally/
#
#  5. Install everything else from this file:
#        pip install -r requirements.txt
#
#  6. Done! Close the command prompt.
#

#  ----------------------------------------------------------------

#  - SyPy3                  →  Comes with SynthEyes installation

#  - SAM3 / Qwen2 runners   →  Your own local scripts

#
#  HOW TO INSTALL (One-Click):
#  -----------------------------------------------
#  After cloning the repo, just run the bundled installer from the repo root:
#        python install.py
#
#  That will create .venv (if missing), install PyTorch + this file, and
#  download the required model weights (.safetensors / .pt).
#
#  Manual install (advanced):
#  -----------------------------------------------
#  1. Open a terminal and `cd` into the cloned repo folder (wherever it is).
#  2. Create and activate a venv:
#        python -m venv .venv
#        (Windows) .venv\Scripts\activate
#        (macOS/Linux) source .venv/bin/activate
#  3. Install PyTorch (CUDA 12.1 wheel):
#        pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
#     NOTE: cu121 = CUDA 12.1. If your GPU driver supports a different
#     CUDA version, check https://pytorch.org/get-started/locally/
#  4. Install everything else from this file:
#        pip install -r requirements.txt
#
#  WHAT IS NOT IN THIS FILE (installed separately / already local):
#  ----------------------------------------------------------------
#  - PyTorch + TorchVision  →  Step 4 above (special CUDA index)
#  - CoTracker3 repo        →  Already in your thirdparty/ folder
#  - SyPy3                  →  Comes with SynthEyes installation
#  - core/ package          →  Your own local code
#  - SAM3 / Qwen2 runners   →  Your own local scripts
# =============================================================
